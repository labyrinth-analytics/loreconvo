Debbie's 4 Decision Paths (Phase 3 → 4)

1. Ignore

  1. Leave item in raw/
  2. No action taken
  3. Auto-pruned at 90 days
  4. Most agent reports end here

2. Need More Info

  1. Ask question in Cowork session
  2. OR: drop question file in raw/inbox/ with agent: gina
  3. Agent responds via their own raw/ folder, tagged review:debbie
  4. Jacqueline surfaces response in next dashboard
  5. Return to decision loop when answered

3. Promote to Processing

  1. Run promote CLI (or git mv)
  2. Assign ID + type (DEC/SPEC/ADR)
  3. Fill in decision context + options
  4. Finalize conclusion
  5. Promote to archive when stable

4. Direct to Archive

  1. Conclusion is already clear
  2. Run promote CLI with --to archive
  3. Skips processing/ entirely
  4. Use for historical decisions already made
